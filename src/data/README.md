# src/data/

This directory is populated at build time by `scripts/sync-data.py`.

**Do not commit real pipeline data here.** The `.gitignore` excludes all
`.json` and `.md` files in this directory.

When `src/data/` is empty (fresh checkout), all pages fall back to
built-in mock data and render correctly without errors.
